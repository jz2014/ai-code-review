# AI Code Review Methodology

Perspective-driven code review methodology that catches bugs AI reviews miss.

After Claude Code and Codex review your code, run this methodology on top — systematic perspective rotation finds what single-perspective AI passes overlook.

- **Claude Code**: reference in `CLAUDE.md`
- **Cursor**: reference in `.cursorrules`
- **Codex**: include in system instructions
- **Human reviewers**: use the checklists as a review guide
